### The List Element

#### Example Usage

```html
<!-- Unordered list items creates bullet points -->
<ul>
  <li>List Item 1</li>
  <li>List Item 2</li>
  <li>List Item 3</li>
</ul>

<!-- Ordered list items creates numbered points -->
<ul>
  <li>List Item 1</li>
  <li>List Item 2</li>
  <li>List Item 3</li>
</ul>
```

### Nesting and Indentation

#### Example Usage

```html
<!-- You can nest list items to create sub points -->
<ul>
  <li>List Item 1</li>
  <li>List Item 2</li>
  <ul>
    <li>Sub Item 1</li>
  </ul>
</ul>
```
